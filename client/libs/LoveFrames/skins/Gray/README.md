# Gray Theme
## Information
Gray is a theme for [Love Frames](https://github.com/NikolaiResokav/LoveFrames) - GUI library for [LÃ–VE](http://www.love2d.org).

## Installation
1. Put **Gray** folder into skins/ folder inside Love Frames.
2. Open **init.lua** in root Love Frames folder and set ```loveframes.config["ACTIVESKIN"]``` to ``"Gray"``.
